Git操作远程仓库

作者:Adiana    发布于: 2018-03-02

方法一

  • 新建文件夹
  • git config –global user.name
  • git config –global user.email
  • 确认配置信息 git config –list
  • 生成公钥 ssh-keygen -t rsa -C “email”
  • 把公钥推到github.com
  • git clone https/ssh

    方法二

  • 新建文件夹
  • git init
  • git add .
  • git remote add origin 远程地址
  • git fetch origin 分支名
  • git branch 新分支 本地新建分支
  • git checkout 分支 切换本地分支
  • git push origin 本地分支 将本地分支推到远程 origin为远程分支的默认名称
    • git push origin 本地分支:master(远程分支）
  • git pull origin 远程分支 拉取远程分支
